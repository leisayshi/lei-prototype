Create Coupon
📋 Basic Info
* Name
Customer-facing name shown on invoices/receipts. Max 40 characters.
* Discount Type
* Percentage
%
Enter a value between 1 and 100. 100% means the order is completely free.
* Amount
+ Add currency
Specify the discount amount per currency. Leave currencies unconfigured to exclude them.
⏱️ Duration & Limits
* Duration
Determines how many billing cycles the discount applies to.
* Duration Months
Number of billing cycles the discount will apply. Only relevant for subscriptions.
Max Redemptions
Total number of times this coupon can be redeemed across all promotion codes.
Daily Redemption Limit
Optional. Limit how many times a single customer can redeem per day.
Expiry Date
Optional. After this datetime, the coupon cannot be redeemed.
🎯 Applicable Scope
Category Tags
gaming × subscription ×
Only orders with matching goodsCategory will be eligible. Leave empty for all categories.
Metadata
+ Add field
Custom key-value pairs for your own tracking purposes.
